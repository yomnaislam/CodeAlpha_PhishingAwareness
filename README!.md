# CodeAlpha - Phishing Awareness Training

## Task 2 – CodeAlpha Cyber Security Internship

This project was developed as part of the **CodeAlpha Cyber Security Internship**.

The purpose of this project is to educate users about phishing attacks, common social engineering techniques, and best practices for staying safe online.

---

## Project Objectives

- Explain what phishing is
- Describe different types of phishing attacks
- Identify fake websites and phishing emails
- Explain social engineering techniques
- Provide cybersecurity best practices
- Test knowledge through a short quiz

---

## Topics Covered

- What is Phishing?
- Types of Phishing
- Fake Websites and Email Tricks
- Social Engineering
- Best Practices for Protection
- Interactive Quiz

---
